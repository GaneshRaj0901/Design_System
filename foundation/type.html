Typography

The GR type system uses two typefaces: Fraunces for display headings and Work Sans for body copy and UI text. Together they create a clean, modern, and highly legible hierarchy.

h1 · Fraunces · 3rem · 900 · Display

Page Title

Use the Display heading for the single most important text element on a page — a hero headline or the page title. It should appear only once per page. The heavy weight of Fraunces commands immediate attention and establishes brand personality.

h2 · Fraunces · 2rem · 900 · Section Header

Section Header

Use H2 to label major sections of a page such as Color, Typography, or Components. It creates a clear visual break between content zones. Keep H2 headings short — they are navigational anchors as much as they are titles.

h3 · Work Sans · 1.5rem · 600 · Subsection Title

Subsection Title

H3 is the first heading level set in Work Sans, signaling a shift from brand display type to functional UI type. Use it for named subsections within a larger section — for example, "Font Pairing" under Typography. It pairs naturally with body text below it.

h4 · Work Sans · 1.25rem · 600 · Card Title

Card Title

H4 is used for card titles, list item labels, and component names throughout the design system. It is compact enough to fit inside a card without overwhelming the description text below it. Use sentence case for H4 labels.

p · Work Sans · 1rem · 400 · Body

This is body copy set in Work Sans Regular at 16px. It is the primary reading size for all paragraphs, descriptions, and explanatory text. Line height is set to 1.7 for comfortable reading across all screen sizes. Body text should never compete visually with headings — let the type scale do the work.

small · Work Sans · 0.875rem · 400 · Caption / Label

Caption text · Labels · Metadata · Type spec annotations

Use small text for captions beneath images, technical labels under type specimens (as seen above), timestamps, and secondary metadata. It should never be used for primary content — it is always supplementary to a heading or body paragraph above it.
