Thumbnail Card

The thumbnail card displays a single project as a full-width image with a title and short description below it. One card is shown per row, giving each project room to breathe. There is no button — the card itself is the entry point, keeping the layout clean and focused on the visual content.

Accessibility

  • The card image must have a descriptive alt attribute that communicates what the project shows — not a generic label.
  • Use semantic HTML — a <div class="card"> with a <h4> title and <p> description below the image.
  • If the card is a link, wrap it in an <a> element with a visible focus ring — do not rely on click events on non-interactive elements.
  • Ensure the title text has sufficient contrast against the card background to meet WCAG AA.

Details of Use

  • One card per row — the image spans the full content width.
  • Card anatomy: full-width project image on top, H4 title below, followed by a short description.
  • No CTA button — the card layout itself guides the user to the project.
  • Keep the title concise and the description to two lines maximum to maintain consistent card height across the page.

Code

<div class="card border-0 shadow-sm" style="border-radius:12px; overflow:hidden;">
  <img src="images/project-thumbnail.jpg"
    class="card-img-top img-fluid"
    alt="Descriptive alt text for the project image">
  <div class="card-body p-4">
    <h4 class="card-title mb-2"
      style="font-family:'Work Sans',sans-serif; font-weight:600; color:#1A1A2E;">
      Project Title
    </h4>
    <p class="card-text mb-0"
      style="font-family:'Work Sans',sans-serif; color:#666; font-size:0.9rem;">
      A brief description of this project.
    </p>
  </div>
</div>
