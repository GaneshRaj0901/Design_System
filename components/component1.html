Navigation Bar

The GR navigation bar is a minimal, full-width bar that sits at the top of every page. The logo mark is anchored to the left, and three text links — Home, About, and Resume — are spaced evenly on the right. The active page is indicated by bold weight. The design is intentionally light, using a white background and dark text to keep focus on the content below.

Accessibility

  • Wrap the navbar in a <nav> element with aria-label="Main navigation" so screen readers identify it as a landmark region.
  • All links must be keyboard-navigable via Tab and show a visible focus state — do not suppress the browser default outline without replacing it.
  • The active page link must include aria-current="page" to communicate location to assistive technologies.
  • Maintain sufficient contrast between the nav text and background — dark text on white exceeds WCAG AA at all sizes.

Details of Use

  • Place on every page at the top, sticky or static.
  • Logo anchored to the far left; links (Home, About, Resume) on the far right with generous spacing between them.
  • Active page link is set in bold weight — no underline or color change.
  • On mobile, links collapse into a hamburger menu.

Code

<nav class="navbar navbar-expand-md" aria-label="Main navigation"
  style="background:#1A1A2E; padding:0 1.5rem;">

  <a class="navbar-brand fw-bold" href="#"
    style="font-family:'Fraunces',sans-serif; color:#0E6DF9;">G·R</a>

  <button class="navbar-toggler border-0" type="button"
    data-bs-toggle="collapse" data-bs-target="#demoNav"
    aria-controls="demoNav" aria-expanded="false"
    aria-label="Toggle navigation">
    <span class="navbar-toggler-icon"></span>
  </button>

  <div class="collapse navbar-collapse justify-content-end" id="demoNav">
    <ul class="navbar-nav gap-3">
      <li class="nav-item">
        <a class="nav-link" href="#" aria-current="page"
          style="color:#fff; border-bottom:2px solid #F5A623;">Work</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#"
          style="color:rgba(255,255,255,0.7);">About</a>
      </li>
      <li class="nav-item">
        <a class="nav-link" href="#"
          style="color:rgba(255,255,255,0.7);">Contact</a>
      </li>
    </ul>
  </div>
</nav>
